# Valentine's Day Special Website

This is a beautiful, mobile-responsive website designed for Valentine's Day. It features a romantic hero section with floating hearts, a memory gallery, and an interactive love letter.

## Features
- **Floating Hearts Background**: Pure CSS animation.
- **Interactive Envelope**: Click to open and reveal a custom letter.
- **Memory Gallery**: Placeholders for your favorite photos.
- **Mobile Responsive**: Looks great on phones and desktops.

## How to Customize

1.  **Change the Letter**:
    - Open `index.html`.
    - Find the `<div class="text">` inside the `letter` section.
    - Edit the text inside the `<p>` tags to write your own message.

2.  **Add Your Photos**:
    - Save your photos in the project folder (or an `images` subfolder).
    - Open `index.html`.
    - Replace the `src="https://placehold.co/..."` in the `gallery-item` divs with your image filenames (e.g., `src="photo1.jpg"`).

3.  **Add Background Music**:
    - Put an MP3 file named `music.mp3` in this folder.
    - The play button will automatically work.

4.  **Edit Animations**:
    - You can change colors and speeds in `style.css`.

## Running the Site
Simply open `index.html` in your web browser.
